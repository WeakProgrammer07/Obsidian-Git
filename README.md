# Obsidian Vault
This repository contains personal notes synchronized using the Obsidian Git extension.

## Overview
The vault uses Git for version control and automated backups. All notes are written in Markdown and are accessible through any standard text editor or the Obsidian application.

## Core Features
- Automated commits and pushes at set intervals.
- Conflict resolution through Git version history.
- Local-first storage with cloud redundancy.
- Full history of changes for all documents.
